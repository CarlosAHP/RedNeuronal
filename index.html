06

Modelo McCulloch-Pitts

La primera neurona artificial (1943)

Principio "Todo o Nada"

Entradas Excitatorias (0 o 1)
Entradas Inhibitorias (0 o 1)
Umbral
Salida Binaria (0 o 1)

Reglas

1 Si hay entrada inhibitoria activa → salida = 0
2 Si suma excitatoria ≥ umbral → salida = 1
Limitación: Solo funciones lógicas básicas, difícil de entrenar
07

Programación McCulloch-Pitts

Implementación en Python

Función neurona_mip

def neurona_mip(entradas_excitatorias, 
                 entradas_inhibitorias, 
                 umbral):
    # Regla 1: Inhibición
    if any(entradas_inhibitorias):
        return 0
    
    # Regla 2: Excitación
    suma = sum(entradas_excitatorias)
    return 1 if suma >= umbral else 0

Ejemplos

Operación NOT

Entrada: 1 → Salida: 0

Operación OR

Entradas: [1,0] → Salida: 1

08

El Perceptrón

Frank Rosenblatt (1950s)

Fórmula Matemática

y = f(W·x + b)
x: vector de entradas
W: vector de pesos
b: sesgo
f: función de activación (escalón)

Limitación

Solo puede resolver problemas linealmente separables

Problema XOR

Un perceptrón simple NO puede resolver XOR

Solución: Combinar múltiples perceptrones en capas
09

Del Perceptrón a la Red Neuronal

Transición para entrenamiento automático

Problema

Función escalón no es derivable → No se puede usar descenso por gradiente

Solución

Reemplazar escalón por función sigmoide (derivable y continua)

Función Sigmoide

Fórmula: 1/(1+e^(-x))
Rango: [0, 1]
Ventaja: Suave y derivable
10

Funciones de Activación

Romper la linealidad en redes profundas

Sigmoide

1/(1+e^(-x)) → [0,1]
✓ Suave y derivable ✗ Desvanecimiento del gradiente

Tanh

(e^x - e^(-x))/(e^x + e^(-x)) → [-1,1]
✓ Mejor que sigmoide ✗ También sufre desvanecimiento

ReLU

max(0, x)
✓ Mitiga desvanecimiento ✓ Entrenamiento más rápido
11

Cálculo de Salida

Ejemplo paso a paso

Datos del Ejemplo

Entradas: x₁=8, x₂=4
Pesos: w₁=0.3, w₂=0.9
Sesgo: b=5

Pasos de Cálculo

1
Calcular H (logit):
H = w₁×x₁ + w₂×x₂ + b
H = 0.3×8 + 0.9×4 + 5 = 11.0
2
Aplicar función sigmoide:
y_pred = 1/(1+e^(-H))
y_pred = sigmoide(11.0) ≈ 0.999
12

Descenso por Gradiente

Algoritmo fundamental (Cauchy, 1847)

Algoritmo

1 Inicialización: Valores aleatorios
2 Cálculo del gradiente
3 Actualización: Moverse en dirección opuesta
4 Tasa de aprendizaje controla el paso

Tasa de Aprendizaje

Muy pequeña: Aprendizaje lento
Muy grande: Divergencia (oscilación)
Óptima: Convergencia eficiente

Cálculo Multicapa

Ejemplo paso a paso

Datos

Cálculo por Capas

Capa Salida:

H₂ = A₁·W₂ + B₂ = -2.258
y_pred = tanh(H₂) = -0.978
